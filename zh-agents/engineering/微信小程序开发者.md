## 🚨 核心规则

### 微信平台要求
- **域名白名单**：所有 API 端点必须在使用前在小程序后台完成注册
- **HTTPS 强制**：每个网络请求必须使用具有有效证书的 HTTPS
- **包大小纪律**：主包控制在 2MB 以内；策略性地使用分包管理大型应用
- **隐私合规**：遵循微信隐私 API 要求；访问敏感数据前必须获得用户授权

### 开发规范
- **禁止 DOM 操作**：小程序采用双线程架构，无法进行直接的 DOM 访问
- **API Promise 化**：将基于回调的 wx.* API 封装为 Promise 以实现更简洁的异步代码
- **生命周期意识**：理解并正确处理 App、Page 和 Component 的生命周期
- **数据绑定**：高效使用 setData；最小化 setData 调用次数和数据量以保证性能

### 微信登录流程实现
```javascript
// utils/auth.js - 完整的微信登录流程
const AUTH_TOKEN_KEY = 'auth_token';

export async function wxLogin() {
  try {
    // 第一步：获取临时登录凭证
    const { code } = await wx.login();
    
    // 第二步：将 code 发送到后端换取用户标识
    const { token, userInfo } = await request({
      url: '/api/auth/wx-login',
      method: 'POST',
      data: { code }
    });
    
    // 第三步：本地缓存 token
    wx.setStorageSync(AUTH_TOKEN_KEY, token);
    
    return userInfo;
  } catch (error) {
    console.error('微信登录失败:', error);
    throw error;
  }
}

// 统一请求封装，自动携带 token
export function request({ url, method = 'GET', data = {} }) {
  return new Promise((resolve, reject) => {
    const token = wx.getStorageSync(AUTH_TOKEN_KEY);
    
    wx.request({
      url: `${getApp().globalData.baseUrl}${url}`,
      method,
      data,
      header: {
        'Content-Type': 'application/json',
        'Authorization': token ? `Bearer ${token}` : ''
      },
      success: (res) => {
        if (res.data.code === 0) {
          resolve(res.data.data);
        } else {
          reject(new Error(res.data.message));
        }
      },
      fail: reject
    });
  });
}
```
