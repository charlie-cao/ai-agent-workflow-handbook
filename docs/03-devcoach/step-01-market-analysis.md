# Step 1：市场分析

> Agent 使用：**市场趋势研究员**
> 时间：30-60 分钟
> 输出：市场分析报告（痛点、竞品不足、差异化方向、定价建议）

---

## 背景

在写任何代码之前，先用「市场趋势研究员」Agent 验证：  
**这个市场值得做吗？我们的差异化在哪里？**

---

## 使用的 Agent

**市场趋势研究员**（Market Trend Researcher）

- 激活方式：Copilot Chat Agent 模式 → 选「市场趋势研究员」
- 专长：市场情报、竞品分析、机会识别

---

## 完整提示词（可直接复用）

```
激活 市场趋势研究员

我要做一个面向程序员的 AI 学习 SaaS，帮助他们通过 AI 辅导快速提升编程能力。
目标用户：工作 1-5 年的后端/全栈工程师，想系统提升架构/算法/某语言能力。
竞品：Educative、Scrimba、LeetCode Premium、GitHub Copilot（间接竞品）。

请分析：
1. 这个市场的痛点和机会在哪里（"AI + 程序员成长"赛道）
2. 竞品在 AI 辅导上的不足
3. 我们可以主打的 2 个差异化功能方向
4. 目标用户愿不愿意付费（付费意愿和定价区间）
```

---

## 实际输出（2026-03-25）

### 核心痛点

- 学了不知道学到什么程度够用（没有基准）
- 写的代码没人 Review，不知道哪里差
- ChatGPT 回答太泛，不够个性化
- 不知道「我现在是什么水平、下一步补什么」

### 竞品不足

| 竞品 | 不足 |
|------|------|
| Educative | 静态内容，无 AI 反馈，无个性化 |
| Scrimba | 仅覆盖前端/初学者，AI 功能浅 |
| LeetCode Premium | 只刷题，架构/工程能力空白 |
| GitHub Copilot | 写代码助手，不是成长辅导 |

**共同盲点**：没有「给某个工程师做能力诊断 → 推荐个性化成长路径」的闭环。

### 差异化方向（MVP 选择）

**方向 A（MVP 先做）：实战代码诊断**
- 用户提交代码 → AI 输出专业 Review
- 不只说「有 bug」，而是「你对并发理解不足，推荐这 3 个练习场景」
- 从一段代码推断能力地图

**方向 B（后续做）：技能水平 GPS**
- 持续追踪工程师的能力图谱（架构/算法/工程实践各维度）
- 可视化成长路径

### 定价建议

| 套餐 | 价格/月 | 内容 |
|------|---------|------|
| 免费版 | ¥0 | 3 次诊断/月，报告前 50% |
| 个人版 | ¥39-59 | 无限诊断 + 完整报告 + AI 对话 |
| Pro | ¥99 | 加能力地图追踪 + 个性化路径 |

海外定价：$15-29/月
