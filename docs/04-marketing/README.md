## 营销总策略

**产品**：DevCoach — 面向程序员的 AI 技能成长 SaaS

**目标用户**：工作 1-5 年、主动追求成长的后端/全栈工程师

**核心信息（所有平台统一）**：
> "你写过成千上万行代码，但你真的知道自己哪里不够好吗？DevCoach 用 AI 帮你看清成长盲区。"

**各平台策略**：

| 平台 | 内容类型 | 目标 | Agent |
|------|----------|------|-------|
| 微信公众号 | 深度技术文章 + 工具推荐 | 建立信任 + 转化课程 | 微信公众号运营专家 |
| 朋友圈 | 用户案例 + 产品更新 | 口碑传播 | 内容创作者 |
| 抖音 | 「程序员进阶技巧」系列短视频 | 曝光 + 下载 | 抖音运营策略师 |
| 小红书 | 「工程师成长攻略」图文 | 种草 + 口碑 | 小红书运营专家 |
| Facebook | 精准定向广告（海外开发者）| 付费获客 | 付费社交策略师 |

## 如何使用 Agent 生成营销内容

```
# 生成微信公众号文章
激活 微信公众号运营专家

产品：DevCoach，AI 驱动的程序员技能成长工具
目标读者：工作 3 年的后端工程师，担心被 AI 替代、想快速提升
文章方向：[你的想法，如"AI 时代，程序员最需要提升什么能力"]
```

```
# 生成抖音脚本
激活 抖音运营策略师

产品介绍：DevCoach，提交代码 → AI 给你专业评审和成长建议
目标：吸引工程师下载试用
视频时长：60 秒以内
风格：干货向/实用主义，不要煽情
```
